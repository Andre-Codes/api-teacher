pandas.DataFrame.agg

The pandas.DataFrame.agg function is used to aggregate using one or more operations over the specified axis.

Parameters

Name Description
func A function, string, or list of functions to use for aggregation.
axis The axis to apply the aggregation on. Default is 0 (index).
*args Additional positional arguments to pass to the aggregation functions.
**kwargs Additional keyword arguments to pass to the aggregation functions.

Returns

Returns a new DataFrame with the aggregated results.

Code Examples

Example 1:


import pandas as pd

# Create a sample DataFrame
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}
df = pd.DataFrame(data)

# Aggregate using sum function
result = df.agg('sum')

print(result)

Output:


A    6
B    15
dtype: int64

Example 2:


import pandas as pd

# Create a sample DataFrame
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}
df = pd.DataFrame(data)

# Aggregate using multiple functions
result = df.agg(['sum', 'mean'])

print(result)

Output:


     A    B
sum  6   15
mean 2.0 5.0

Example 3:


import pandas as pd

# Create a sample DataFrame
data = {'A': [1, 2, 3], 'B': [4, 5, 6]}
df = pd.DataFrame(data)

# Aggregate using a custom function
def custom_agg(x):
    return x.max() - x.min()

result = df.agg(custom_agg)

print(result)

Output:


A    2
B    2
dtype: int64
