transform vs apply method in pandas

The transform and apply methods in pandas are used to perform operations on DataFrame columns or rows. The main difference between them is that transform returns a DataFrame with the same shape as the original, while apply can return a Series or a DataFrame with a different shape.

Parameters

Parameter Description
func The function to apply to each column or row.
axis The axis along which the function is applied. 0 for columns, 1 for rows.

Examples

Example 1:


import pandas as pd

df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# Using transform
df_transformed = df.transform(lambda x: x * 2)
print(df_transformed)

Example 2:


import pandas as pd

df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# Using apply
df_applied = df.apply(lambda x: x * 2)
print(df_applied)

Example 3:


import pandas as pd

df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})

# Using transform with axis=1
df_transformed_rows = df.transform(lambda x: x * 2, axis=1)
print(df_transformed_rows)
