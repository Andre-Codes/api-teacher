Stratified Sampling for Linear Regression with scikit-learn

API Documentation

Parameters

Parameter Description
n_splits Number of re-shuffling & splitting iterations.
test_size Float or int value representing the proportion or absolute number of test samples.
train_size Float or int value representing the proportion or absolute number of train samples.
random_state Seed used by the random number generator.

Code Example


import numpy as np
from sklearn.model_selection import StratifiedShuffleSplit

# Generate some dummy data
X = np.random.rand(100, 5)
y = np.random.randint(0, 2, 100)

# Create the StratifiedShuffleSplit object
stratified_split = StratifiedShuffleSplit(n_splits=3, test_size=0.2, random_state=42)

# Perform stratified sampling
for train_index, test_index in stratified_split.split(X, y):
    X_train, X_test = X[train_index], X[test_index]
    y_train, y_test = y[train_index], y[test_index]
