pandas to_numeric function

The to_numeric function in pandas is used to convert the values in a given Series or DataFrame to numeric type.

Parameters:

Parameter Description
arg The object to convert to a numeric type.
errors Specifies how to handle errors during the conversion. Possible values are 'raise', 'coerce', and 'ignore'.
downcast Specifies a dict of item->dtype of what to downcast if possible.

Returns:

Returns a numeric type (int, float, or complex) or a Series or DataFrame with the numeric type.

Code Examples:

Example 1: Convert a Series of strings to numeric type:


import pandas as pd

# Create a Series with strings
s = pd.Series(['1', '2', '3', '4', '5'])

# Convert the Series to numeric type
s_numeric = pd.to_numeric(s)

print(s_numeric)

Output:


0    1
1    2
2    3
3    4
4    5
dtype: int64

Example 2: Convert a DataFrame column to numeric type, ignoring errors:


import pandas as pd

# Create a DataFrame
df = pd.DataFrame({'A': ['1', '2', '3', '4', '5'],
                   'B': ['6', '7', '8', '9', '10']})

# Convert column 'A' to numeric type, ignoring errors
df['A'] = pd.to_numeric(df['A'], errors='ignore')

print(df)

Output:


   A   B
0  1   6
1  2   7
2  3   8
3  4   9
4  5  10

Example 3: Convert a DataFrame column to numeric type, raising errors:


import pandas as pd

# Create a DataFrame
df = pd.DataFrame({'A': ['1', '2', '3', '4', '5'],
                   'B': ['6', '7', '8', '9', '10']})

# Convert column 'A' to numeric type, raising errors
df['A'] = pd.to_numeric(df['A'], errors='raise')

print(df)

Output:


   A   B
0  1   6
1  2   7
2  3   8
3  4   9
4  5  10

In the first example, we create a Series with strings and use the to_numeric function to convert it to a numeric type. The resulting Series contains integers.

In the second example, we create a DataFrame with two columns and convert column 'A' to numeric type, ignoring errors. The resulting DataFrame remains the same.

In the third example, we create a DataFrame with two columns and convert column 'A' to numeric type, raising errors. Since the values in column 'A' are valid numbers, the resulting DataFrame remains the same.
