Creating a Lambda Function to Convert Every Other Letter to Uppercase in a DataFrame Column

To achieve this, we can use the apply function in pandas along with a lambda function. The lambda function will iterate over each character in the column and convert every other letter to uppercase.

Code


import pandas as pd

# Sample DataFrame
df = pd.DataFrame({'column_name': ['hello', 'world', 'example']})

# Lambda function to convert every other letter to uppercase
df['column_name'] = df['column_name'].apply(lambda x: ''.join([c.upper() if i % 2 == 1 else c for i, c in enumerate(x)]))

Explanation

The code above uses the apply function to apply a lambda function to each element in the 'column_name' column of the DataFrame. The lambda function iterates over each character in the string and checks if the index is odd (i % 2 == 1). If the index is odd, it converts the character to uppercase using the upper() method; otherwise, it keeps the character as it is. Finally, the lambda function joins the modified characters back into a string.

Examples

Input Output
'hello' 'hElLo'
'world' 'wOrLd'
'example' 'eXaMpLe'
